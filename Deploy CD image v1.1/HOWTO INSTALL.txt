RAMQ FileConverter installation procedure
-----------------------------------------

The RAMQ FileConverter has to be manually installed on the target computer.

1. Copy the directory named "RAMQ FileConverter" in C:\Program Files.

2. Go in the "C:\Program Files\RAMQ FileConverter" directory.

3. Right click on the RAMQFileConverter.exe file and select: 
   send to -> Desktop (create shortcut). This action will create a shortcut 
   on the desktop for the RAMQ FileConverter.

To use the RAMQ FileConverter just click on the newly created shortcut placed 
the desktop.
